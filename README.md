## Getting Started

### Prerequisites

1. have node and npm installed

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/NielsDobbelaar/new-personal-website
   ```
2. Install the dependencies
   ```sh
   npm i
   ```
3. Start the development server
   ```sh
   npm run dev
   ```
